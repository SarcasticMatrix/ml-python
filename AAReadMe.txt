Le zip est composé de deux dossiers, un pour le modèle de régression logistique et un pour les modèles de boosting. 

modèle Régression Logistique/data/preparation.py contient les fonctions utilitaires ainsi que la Pipeline de transofrmation des données utilisée dans le cadre de ce modèle
modèle Régression Logistique/data/data_prep.ipynb est le notebook associé à cette transformation des données
modèle Régression Logistique/models/LogReg.ipynb est le notebook associé à l'optimisation du modèle et des tests.
modèle Régression Logistique/data/01_Input_Data/bank-additionnal-full.csv contient les données utilisées dans ce cas

modèles boosting/preparation.py contient les fonctions utilitaires ainsi que la Pipeline de transofrmation des données utilisée dans le cadre de ce modèle
modèles boosting/Exploratory analysis & Feature engineering.ipynb est le notebook associé à cette transformation des données
modèles boosting/modelsA.ipynb est le notebook associé aux modèles de boosting ici
modèles boosting/modelsA.html est la version html
modèle Régression Logistique/data/data.csv contient les données utilisées dans ce cas
ml_python_rapport.pdf est le rapport
